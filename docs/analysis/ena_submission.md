# Submit Analysis data to ENA

1After successful validation you should be allowed to do a submission to ENA. You will need to provide your credentials.
![Screenshot](../img/new_system/analysis/ena_login.png)


2. After submission, the results will be available on the data portal. 
   Your browser should automatically download the submission_results.xml file. 
   If the download does not start automatically, click the "Download submission results" button. 
   It is important to save this XML file in a secure location for future reference.


Please contact [FAANG DCC](mailto:faang-dcc@ebi.ac.uk) if you have any issues with any of the processes on this page.
